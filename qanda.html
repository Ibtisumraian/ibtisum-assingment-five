Question-1: What are the different ways to select an element in the DOM?

Ans : There are many ways by which we can select an element. As like : 1. getElementById() 2. getElementsByClassName() 3. querySelector() 4. querySelectorAll() 5. getElementsByTagName() 6. getElementsByName() 7.etc

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans : innerHTML : We can get or set the HTML content of an element. innerText : We can get or set the Text content of an element. textContent :Including hidden elements we can get or set the Text content of an element .

Question-3: What is event delegation in the DOM?

Ans : Event delegation is a technique where instead of using event listener to multiple child we can use event listener on childs parent. It works because of event bubbling.

Question-4: What is event bubbling in the DOM?

Ans : Event bubbling is a mechanism where, when the event is triggered on a child first it runs it's event handler than bubbles up to its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans : To create elements in JavaScript we can use createElement() and to add elements in JavaScript we can use appendChild() and to remove elements in JavaScript we can use removeChild().
